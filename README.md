## Instructions
- Open `index.html` (recommended: VS Code Live Server)
- Hover points to inspect the behaviour log
- Press `1 / 2 / 3` to switch Day views, `0` for All data, `4` for Live (user-input) events
- Type an action + time (`HH:MM`) + location, then click `LOG EVENT` to generate a new classified entry

## About
Domestic Surveillance is an interactive small-data artwork that maps everyday actions into a pseudo-surveillance interface.  
It explores how ordinary behaviour can appear suspicious once it is logged, classified, and observed by a system.
